# Ford vs Finn: Battle Arena

Two-player tap battle game for Ford (age 7) and Finn (age 9). One iPad, two fighters, one winner.

## How to play

- Ford taps the left side of the screen to attack Finn
- Finn taps the right side to attack Ford
- Each tap deals 3–7 random damage
- First player to reach 0 HP loses
- Tap **PLAY AGAIN** to reset

## Play it

[roller.github.io/ford-battle-arena](https://roller.github.io/ford-battle-arena)

Open in Safari on the iPad → Add to Home Screen → it's an icon.

## Tech

Single `index.html`, no build step, no dependencies. Web Audio API for sword sounds.
